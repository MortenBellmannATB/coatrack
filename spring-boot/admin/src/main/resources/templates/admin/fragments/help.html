To get started with CoatRack, you need to do the following:

  • Define Service Offering: Tell CoatRack about Your Service and how it should be offered (access policies, pricing)
  • Install Service Gateway: Download your custom CoatRack Gateway and install it in your network
  • Generate API keys: Generate API keys for consumers of Your Service

There are different wildcards that you can apply to your pay per call paths, such as:

  • ? : matches one character
  • * : matches zero or more characters
  • ** : matches zero or more directories in the path

For additional information about wildcards and regular expressions:

  • CoatRack uses the AntPathMatcher for the wildcards, you will find more information in the AntPathMatcher API documentation
  • The regular expressions used in CoatRack have to be in Java Style, for more information see related API documentation page
